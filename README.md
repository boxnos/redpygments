## Installation

install pygments

### fedora user
~~~~
$ sudo yum install python-pygments
~~~~

### clone
~~~~
$ git clone https://github.com/boxnos/redpygments.git
~~~~

### install
~~~~
$ cd redpygments
$ rake install
~~~~

### uninstall
~~~~
$ gem uninstall redpygments
~~~~

## Usage

try it.

~~~~
$ redpygments -s template.slim awesome.md -o awesome.html
~~~~

enjoy!
